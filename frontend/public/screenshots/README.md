# Screenshots Directory

This directory contains screenshots of the Fotmate website for use in the "See Fotmate in Action" carousel on the home page.

## Required Screenshots

Please add the following screenshots (PNG format, recommended size: 1920x1080 or similar):

1. **follow-your-team.png** - Screenshot of the team following/dashboard page showing fixtures, results, and team info
2. **advanced-statistics.png** - Screenshot showing match stats, player performance, and analytics
3. **fantasy-dashboard.png** - Screenshot of the FPL dashboard with squad view
4. **ai-transfer-assistant.png** - Screenshot of the AI transfer recommendations interface
5. **fixture-planner.png** - Screenshot of the fixture planner showing difficulty ratings

## Image Guidelines

- **Format**: PNG (with transparency if needed) or JPG
- **Size**: 1920x1080px or similar aspect ratio (16:9)
- **Quality**: High resolution, optimized for web (use tools like ImageOptim)
- **File size**: Keep under 500KB per image for fast loading
- **Naming**: Use lowercase with hyphens (e.g., `follow-your-team.png`)

## How to Add Screenshots

1. Take screenshots of the actual website pages
2. Optimize them for web (compress if needed)
3. Save them in this directory with the exact filenames listed above
4. The carousel will automatically use them when the images exist

If screenshots are not found, the carousel will display a browser window mockup placeholder.
