**Tagline**: (Optional) "Your Fantasy Football Companion"

## Logo Usage

### Primary Logo

**Full Logo (Horizontal)**:
- Icon + Wordmark side by side
- Primary version for headers, navigation, most contexts

**Stacked Logo (Vertical)**:
- Icon above wordmark
- For square contexts, mobile headers, social media

**Icon Only**:
- Standalone icon/symbol
- For favicons, app icons, small spaces

**Wordmark Only**:
- Text only, no icon
- For contexts where icon doesn't fit

### Logo Variations

**Color Variations**:
- **Full Color**: Primary logo with gradient/solid colors
- **White**: For dark backgrounds
- **Black**: For light backgrounds
- **Gradient**: Special contexts, hero sections

### Minimum Size

- **Full Logo**: 120px width minimum
- **Icon Only**: 16px minimum (favicon), 24px recommended
- **Wordmark Only**: 80px width minimum

### Clear Space

Maintain clear space equal to the height of the icon around the logo on all sides.

## Typography

### Primary Font

**Space Grotesk**:
- Headings, body text, UI elements
- Weights: 300, 400, 500, 600, 700
- Style: Regular (no italic/oblique for logo)

### Secondary Font

**JetBrains Mono**:
- Code, data displays, technical content
- Weights: 400, 500, 600

### Logo Typography

- **Font**: Space Grotesk (customized) or custom lettering
- **Weight**: 600-700 (Semi-bold to Bold)
- **Style**: Regular
- **Letter Spacing**: -0.02em to -0.05em

## Usage Contexts

### Website

**Header/Navigation**:
- Full logo (horizontal)
- 120-180px width
- White or full color

**Footer**:
- Full logo (horizontal)
- 100-120px width
- White or full color

**Login/Register Pages**:
- Full logo (horizontal or stacked)
- 150-200px width
- White or full color

### Mobile App

**App Icon**:
- Icon only
- 512x512px (iOS), 1024x1024px (Android)
- Full color

**Splash Screen**:
- Full logo (stacked or horizontal)
- 200-300px width
- White or full color

**Navigation Header**:
- Icon only or full logo
- 40-64px
- White or full color

- Icon only
- 400x400px
- Full color

- Full logo (horizontal)
