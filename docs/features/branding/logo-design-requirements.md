## Logo Requirements

### Design Requirements

1. **Logo Type**
   - Wordmark (text-based logo with "Fotmate")
   - Optional: Icon/symbol that can work standalone
   - Combination mark (icon + wordmark)

2. **Design Elements**
   - Should reflect football/FPL theme
   - Modern, clean, professional
   - Memorable and distinctive
   - Scalable (works at small and large sizes)

3. **Color Options**
   - Full color version
   - Single color version (for monochrome use)
   - White/light version (for dark backgrounds)
   - Black/dark version (for light backgrounds)

4. **Typography**
   - Modern, readable font
   - Should work well at various sizes
   - Consider custom lettering or modified typeface

5. **Icon/Symbol** (Optional)
   - Football-related element
   - Could incorporate:
     - Football/soccer ball
     - Trophy
     - Chart/graph (data/analytics theme)
     - Combination of elements
   - Should work as standalone icon (favicon, app icon)

## Technical Requirements

### File Formats
- **Vector**: SVG, AI, EPS (for scalability)
- **Raster**: PNG (multiple sizes), JPG (if needed)
- **Favicon**: ICO, PNG (16x16, 32x32, 48x48, 64x64, 128x128, 256x256)

### Sizes Needed
- **Large**: 1920x1080px (for hero sections, headers)
- **Standard**: 512x512px (for app icons, social media)
- **Medium**: 256x256px (for headers, navigation)
- **Small**: 128x128px (for mobile navigation)
- **Tiny**: 64x64px (for favicons, small UI elements)
- **Favicon**: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256

### Usage Contexts
- **Website Header**: Horizontal logo (wordmark or combination)
- **Mobile App Icon**: Square icon (symbol or combination)
- **Favicon**: Small icon (symbol or letter "F")
- **Social Media**: Square or horizontal (depending on platform)
- **Email Signatures**: Horizontal logo
- **Print Materials**: High-resolution vector

## Design Guidelines

### Style Preferences
- **Modern**: Contemporary design, not outdated
- **Clean**: Minimal, uncluttered
- **Professional**: Suitable for a data/analytics platform
- **Approachable**: Friendly, not too corporate
- **Football-Inspired**: Subtle football/FPL references (not too literal)

### Color Palette Considerations
- Should work with existing app color scheme:
  - Green (primary)
  - Pink (accent)
  - Cyan (secondary)
  - Purple (tertiary)
  - Dark backgrounds
- Logo should work on:
  - Light backgrounds
  - Dark backgrounds
  - Colored backgrounds
  - Gradient backgrounds

### Typography Considerations
- **Readability**: Must be readable at small sizes
- **Distinctiveness**: Should stand out from competitors
- **Versatility**: Works in horizontal and vertical layouts
- **Character**: "Fotmate" - consider how "F" and "M" work together
