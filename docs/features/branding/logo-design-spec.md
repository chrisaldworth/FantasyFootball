## Overview

This document provides complete design specifications for the Fotmate logo. The logo should represent a Fantasy Football Companion Platform that combines data-driven insights with an approachable, modern aesthetic.

**Brand Name**: Fotmate  
**Domain**: fotmate.com  
**Tagline**: (Optional) "Your Fantasy Football Companion"

---

## Research Summary

### Current Logo Implementation
- **Placeholder**: Gradient "F" logo in rounded square
- **Colors**: Green to cyan gradient (#00ff87 to #00d4ff)
- **Usage**: Header, footer, login pages, favicon
- **Component**: `TeamLogo` component with fallback

### Existing App Design System
- **Colors**: Green (#00ff87), Pink (#e90052), Cyan (#04f5ff), Purple (#9d4edd)
- **Typography**: Space Grotesk (primary), JetBrains Mono (code)
- **Style**: Modern, clean, glass morphism, dark theme
- **Tone**: Professional yet approachable, data-focused

### Design Rationale
The logo should:
- Work seamlessly with existing app colors and design
- Reflect football/FPL theme subtly (not too literal)
- Be modern and professional (data/analytics platform)
- Be memorable and distinctive
- Scale well from 16px (favicon) to 1920px (hero sections)

## Recommended Design: Concept 1 (Modern Wordmark with Icon)

### Primary Logo Design

**Layout**: Horizontal combination mark

**Structure**:
```
[Icon] Fotmate
```

**Icon Design**:
- **Shape**: Stylized "F" or "FM" monogram
- **Style**: Modern, geometric, with subtle football reference
- **Size**: Proportional to text (approximately 1:2 ratio)
- **Details**: 
  - Could incorporate subtle hexagon pattern (football reference)
  - Or subtle chart/graph element (analytics reference)
  - Clean, minimal, recognizable at small sizes

**Wordmark Design**:
- **Typography**: Custom/modified Space Grotesk or similar
- **Weight**: Medium to Bold (600-700)
- **Style**: 
  - "F" and "M" slightly stylized for distinctiveness
  - Rounded corners for friendly feel
  - Professional spacing and kerning
- **Size**: Readable at small sizes (minimum 12px)

**Spacing**:
- Icon to text: 12-16px (scales with logo size)
- Minimum clear space: 1x icon height around logo

## Usage Guidelines

### Minimum Clear Space

**Rule**: Maintain clear space equal to the height of the icon around the logo

**Example**:
- If icon is 40px tall, maintain 40px clear space on all sides
- Never place text, images, or other elements within this space

### Minimum Size

**Full Logo**:
- Minimum width: 120px (desktop)
- Minimum width: 80px (mobile)

**Icon Only**:
- Minimum size: 16px (favicon)
- Recommended minimum: 24px (for readability)

**Wordmark Only**:
- Minimum width: 80px
- Recommended minimum: 100px

### Placement Guidelines

**Do's**:
- ‚úÖ Use full color version on dark backgrounds
- ‚úÖ Use white version on dark/colored backgrounds
- ‚úÖ Use black version on light backgrounds
- ‚úÖ Maintain clear space around logo

## Design Recommendations

### Recommended Approach

**Primary Logo**: Modern wordmark with icon (Concept 1)
- **Icon**: Stylized "F" or "FM" with subtle football reference
- **Wordmark**: "Fotmate" in Space Grotesk (customized)
- **Layout**: Horizontal (icon left, text right)
- **Colors**: Green to cyan gradient for icon, white for text

**Why This Approach**:
1. **Scalable**: Works from 16px (favicon) to 1920px (hero)
2. **Memorable**: Distinctive icon + wordmark combination
3. **Flexible**: Icon can work standalone, wordmark can work alone
4. **Modern**: Contemporary design that fits data/analytics platform
5. **Football-Inspired**: Subtle reference without being too literal
6. **Brand Consistency**: Works with existing app colors and design

### Icon Design Suggestions

**Option A: Stylized "F"**
- Modern, geometric "F"
- Subtle hexagon pattern (football reference)
- Gradient fill (green to cyan)

**Option B: "FM" Monogram**
- Interlocking "F" and "M"
- Football-inspired shape integration
- Gradient fill (green to cyan)

**Option C: Abstract Symbol**
- Football + data visualization element
- Modern, geometric shape
- Recognizable at small sizes
