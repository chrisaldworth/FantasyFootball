# UI Designer Documentation

This folder contains documents created by the UI Designer Agent.

## Handoffs TO UI Designer

Handoffs from PPM to UI Designer are located in feature folders:
- `docs/features/[feature-name]/[feature-name]-handoff-ui-designer.md`

## Design Specifications

Completed design specifications are located in feature folders:
- `docs/features/[feature-name]/[feature-name]-design-spec.md`

## Workflow

1. **Receive Handoff**: Check `docs/features/[feature-name]/*handoff-ui-designer*.md`
2. **Create Design Spec**: Create `docs/features/[feature-name]/[feature-name]-design-spec.md`
3. **Handoff to Developer**: Create `docs/features/[feature-name]/[feature-name]-handoff-developer.md`
