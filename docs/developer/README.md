# Developer Documentation

This folder contains documents created by the Developer Agent.

## Handoffs TO Developer

Handoffs from UI Designer to Developer are located in feature folders:
- `docs/features/[feature-name]/[feature-name]-handoff-developer.md`

## Implementation Documents

- **[Test Failures Handoff](test-failures-handoff-developer.md)** - Test failures to fix
- **[Test Fixes Complete](test-fixes-complete.md)** - Completed test fixes

## Workflow

1. **Receive Handoff**: Check `docs/features/[feature-name]/*handoff-developer*.md`
2. **Implement**: Follow design specifications
3. **Handoff to Tester**: Create `docs/features/[feature-name]/[feature-name]-handoff-tester.md`
