# MVP Roadmap

## Phase 1 — Foundation (Weeks 1–3)
- Frontend skeleton (Next.js)
- Backend API wrapper
- Database + caching layer
- FPL API ingestion + cron jobs

## Phase 2 — Core Tools (Weeks 4–6)
- Team import (via FPL ID)
- Basic team rating
- Basic transfer suggestions
- Fixture difficulty ticker
- Player comparison

## Phase 3 — Polishing (Weeks 6–8)
- Mobile optimisation
- Loading states / skeleton UI
- Branding + landing page
- Basic analytics dashboard
- Free-tier live rank

## Phase 4 — Premium Launch (Weeks 9–12)
- AI transfer assistant
- Captaincy algorithm
- Price change monitor
- Wildcard drafts
- Mini‑league insights
- Stripe subscription

## Phase 5 — Growth (Season Launch)
- SEO content pages
- Email newsletter
- YouTube integration
- Sponsored insights
