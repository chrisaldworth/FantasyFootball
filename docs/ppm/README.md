# Product and Project Manager Documentation

This folder contains documents created by the Product and Project Manager (PPM) Agent.

## Documents

- **[Business Objectives](business-objectives.md)** - Project goals and objectives
- **[Functional Requirements](functional-requirements.md)** - Feature requirements
- **[MVP Roadmap](mvp-roadmap.md)** - Minimum viable product plan
- **[Monetisation Strategy](monetisation-strategy.md)** - Revenue model

## Feature Requirements

Feature-specific requirements are located in `docs/features/[feature-name]/[feature-name]-requirements.md`

## Handoffs

When PPM creates requirements, handoffs are created in the feature folders:
- `docs/features/[feature-name]/[feature-name]-handoff-ui-designer.md` - For UI Designer
- `docs/features/[feature-name]/[feature-name]-handoff-developer.md` - For Developer (after design)
- `docs/features/[feature-name]/[feature-name]-handoff-tester.md` - For Tester (after implementation)
